List Comprehension¶

List comprehension consists of an expression followed byfor statementinside square brackets.

In [5]:
# # create a list with the elements 2**x, where x=0,1,2,
pow = [] 
for x in range(10): 
    pow.append(2 ** x)
print(pow)

[1, 2, 4, 8, 16, 32, 64, 128, 256, 512]
In [9]:
# list comprehension for the above code
p2 = [2**x for x in range(10)]
print(p2)

[1, 2, 4, 8, 16, 32, 64, 128, 256, 512]
In [10]:
# create a list of element as x*x for all even number up to 10
list5=[x*x for x in range(10) if not x%2==0]
list5
Out[10]:
[1, 9, 25, 49, 81]
